Overview

This is an interactive Power BI dashboard designed to visualize and analyze eCommerce data. It provides insights into sales performance, customer behavior, product trends, and revenue metrics. Ideal for business analysts, data enthusiasts, and anyone learning Power BI.

Features

Track sales and revenue over time

Identify top-selling products and categories

Analyze customer trends and segmentation

Interactive filters and slicers for dynamic insights

Visualizations include bar charts, line charts, pie charts, and KPI cards

Dataset

The dashboard uses a sample eCommerce dataset including:

Orders: Order ID, Date, Customer, Product, Quantity, Revenue

Customers: Customer ID, Name, Region, Segment

Products: Product ID, Category, Subcategory, Price

Tech Stack / Tools

Power BI Desktop

Sample eCommerce dataset (CSV or Excel)

How to Use

Download or clone the repository.

Open the .pbix file in Power BI Desktop.

Explore the interactive dashboards using slicers and filters.

Connect your own eCommerce dataset if needed.
